# Poolballz Roadmap

## Phase 1: Local MVP

**Goal:** Make Poolballz usable locally as a web app.

**Mode:** mvp

- Build the directory, filters, detail panel, add venue form, open-source map pins, and user report workflow.
- Persist venues and reports in localStorage.
- Verify with automated tests, production build, and browser checks.

## Phase 2: Shared Data

**Goal:** Move from local-only data to shared community submissions.

- Add backend persistence.
- Add duplicate detection and basic moderation.
- Keep community reports separate from provider/imported facts.

## Phase 3: Mobile Field Use

**Goal:** Improve the phone experience for players arriving in a city.

- Polish geolocation, pinning, nearby sorting, and quick crowd updates.
- Add confidence/freshness signals for reports.
